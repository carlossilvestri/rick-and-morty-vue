## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run tests with Jest framework
```
npm run test
```
### Run tests on development mode
```
npm run test:watch
```

### Lints and fixes files
```
npm run lint
```
